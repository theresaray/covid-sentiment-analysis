# covid-sentiment-analysis-tfidf-bert
Repositori ini berisi analisis sentimen data Twitter terkait COVID-19 dengan fokus pada fenomena panic buying. Metode yang digunakan meliputi TF-IDF dengan model machine learning (LR, NB, SVM) serta BERT. Hasil menunjukkan SVM memberikan performa terbaik dibandingkan BERT pada dataset ini.
